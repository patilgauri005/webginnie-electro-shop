# MERN Stack eCommerce Website

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Vercel Deployment

## Features
- User Registration & Login
- Product Listing
- Shopping Cart
- Order Management
- Responsive UI

## Description
A full-stack eCommerce web application built using the MERN stack. 
The project includes frontend development using React, backend API 
development using Node.js and Express, database management with MongoDB, 
and deployment using Vercel.

Developed during Full Stack Internship at Web Ginnie Solutions.
